# Conversor de Monedas - Alura Challenge

¡Bienvenido/a! Este es mi proyecto del **Conversor de Monedas**, desarrollado para el programa **Oracle Next Education (ONE)** con **Alura LATAM**.

## 🚀 Funcionalidades
* **Conversión en Tiempo Real:** Tasas actualizadas mediante la API de **ExchangeRate-API**.
* **Menú Interactivo:** Opciones fáciles para convertir entre USD, COP y más.

## 🛠️ Tecnologías
* **Java 25** (Uso de Records y HttpClient).
* **Gson 2.13.2** (Para procesar el JSON de la API).
* **IntelliJ IDEA**.
